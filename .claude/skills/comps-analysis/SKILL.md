# Comps Analysis Skill

## Workflow

1. 调 `GET /api/valuation/comps/{code}` 取同行业 PE/PB/PS/ROE 表。
2. 按下面结构输出。

## Output template

```
# {名称}（{代码}）同行业可比分析（draft）

## 1. 行业归属
- 行业：{industry}
- 样本数：{count}

## 2. 目标公司核心指标
| 指标 | 数值 | 行业中位 | 相对位置 |
| --- | --- | --- | --- |
| PE(TTM) | … | … | … |
| PB | … | … | … |
| PS | … | … | … |
| ROE | … | … | … |

## 3. 横向对标解读（≤ 4 条）
- 估值高/低于行业中位 X%
- ROE 高/低于行业中位 → 是否解释估值溢价
- 警告：若 PE 高、ROE 反而低 → 估值溢价缺乏盈利支撑

## 4. 相似度筛选建议
列出 3-5 家在以下维度最接近的：
- 营收规模相近
- ROE 同档
- 业务结构相似（人工补充，模型无法直接判定）

## 5. 局限性（必写）
- 行业划分是粗粒度
- 横截面静态指标，不反映周期位置
- 未做规模/会计准则/分红政策调整

---
免责声明：相对估值仅作研究参考（draft），不构成投资建议。
```
